# CookieFarm

CookieFarm is an A/D framework focused on operations:

- write Python exploits
- run them with `ckc`
- centralize submission and monitoring with `cks`

## Recommended competition usage

1. prepare server/shared config before kickoff
2. validate each exploit with `ckc exploit test`
3. switch to `ckc exploit run` with gradual tuning
4. monitor dashboard and react quickly to deny/error spikes
