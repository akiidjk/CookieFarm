# Setup enviroments for development

## Common

Need to install: 

- [Just](https://just.systems/man/en/introduction.html) (Make but better)
- Python3 (For some script and devs tools)
- Python3 venv (For isolating Python dependencies)
- Bun/Node (For frontend development and some tools)
- Docker (For running local services)
- Git (For version control)

## DevOps

- [Act](https://github.com/nektos/act) locally run GitHub Actions

## Server

- Go v1.26.0
- [Air](https://github.com/air-verse/air) (Live reload for Go)
- golangci-lint (For linting Go code)
- gofumpt (For formatting Go code)
- goreleaser (For building and releasing Go binaries)
- [sqlc](https://docs.sqlc.dev/en/stable/overview/install.html) (For generating Go code from SQL queries) 
- [gotestsum](https://github.com/gotestyourself/gotestsum) (For running Go tests better)

## Client

- Python3.14
- [UV](https://docs.astral.sh/uv/) (For running Python scripts and tools)
- [Ruff](https://docs.astral.sh/ruff/) (For linting/formatting Python code)

## Frontend

- uglify-js (For minifying JavaScript code)
- tailwindcss (For styling the frontend)
